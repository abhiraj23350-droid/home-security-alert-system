## âš™ï¸ Working Principle
The Ultrasonic Sensor sends out ultrasonic sound waves and waits for the echo to return.

1. The sensor continuously measures the distance.
2. If the detected distance is less than a predefined limit (e.g., 20 cm),
3. The Arduino activates the buzzer.
4. If no object is detected within range, the buzzer remains OFF.

This allows the system to detect intruders near doors, windows, or restricted areas.
